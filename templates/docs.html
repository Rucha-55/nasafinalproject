Getting Started

Quick Start Guide

  1. Upload Satellite Image: Navigate to the Analysis page and upload your satellite imagery (PNG, JPG, JPEG formats supported).
  2. AI Processing: Our TensorFlow model will automatically segment the image into different land use categories.
  3. View Results: Explore the analysis results in the Summary, Sample Patches, and Comparison tabs.
  4. Generate Predictions: Use the prediction modules to analyze urban growth, environmental impact, and more.

Pro Tip

For best results, use high-resolution satellite images with clear land use distinctions. Images should be at least 256x256 pixels for optimal segmentation accuracy.

Image Analysis

Land Use Categories

Water Bodies

Rivers, lakes, oceans, and other water features

Buildings

Residential, commercial, and industrial structures

Roads

Streets, highways, and transportation infrastructure

Vegetation

Forests, parks, agricultural land, and green spaces

Land

Bare soil, construction sites, and undeveloped areas

Analysis Features

  • Real-time Segmentation: AI-powered image segmentation using deep learning
  • Percentage Calculation: Automatic calculation of land use distribution
  • Visual Overlay: Color-coded segmentation mask overlay on original image
  • Sample Extraction: Automatic extraction of representative patches for each category

API Reference

  "analysis_type": "full"

Troubleshooting

Upload Issues

Problem: Image upload fails or takes too long

Solution: Ensure image is under 10MB and in supported format (PNG, JPG, JPEG)

Low Accuracy Results

Problem: Segmentation results appear inaccurate

Solution: Use higher resolution images with clear contrast between land use types

Prediction Errors

Problem: AI predictions fail to generate

Solution: Check internet connection and ensure valid analysis data exists
